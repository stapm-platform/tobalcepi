# Fast matching

A faster version of \`

## Usage

``` r
x %fin% table
```

## Arguments

- x:

  values to be matched

- table:

  values to be matched against

## Value

Logical indicating match or not

## Details

See \[fastmatch::fmatch()\]

## Examples

``` r

if (FALSE) { # \dontrun{

x %fin% y

} # }


```
