# Tobacco relative risks

Tobacco relative risks

## Usage

``` r
tobacco_relative_risks
```

## Format

A data table containing the relative risks of 52 disease categories for
current vs. never smokers Webster et al. (2018) .

## References

Webster L, Angus C, Brennan A, Gillespie D (2018). “Smoking and the
risks of adult diseases.”
[doi:10.15131/shef.data.7411451](https://doi.org/10.15131/shef.data.7411451)
.
