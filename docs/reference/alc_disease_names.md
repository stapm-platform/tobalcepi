# Names of alcohol-related diseases

Names of alcohol-related diseases

## Usage

``` r
alc_disease_names
```

## Format

A character vector containing the names of the 48 disease categories of
alcohol related diseases Angus et al. (2018) .

## References

Angus C, Henney M, Webster L, Gillespie D (2018). “Alcohol-attributable
diseases and dose-response curves for the Sheffield Alcohol Policy Model
version 4.0.” The University of Sheffield.
<https://doi.org/10.15131/shef.data.6819689>.
